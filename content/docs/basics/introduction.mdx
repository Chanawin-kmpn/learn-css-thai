# CSS คืออะไร

CSS ย่อมาจาก Cascading Style Sheets ซึ่งเป็นภาษาที่ใช้สำหรับการออกแบบหน้าเว็บ โดย CSS จะช่วยให้เราสามารถกำหนดรูปแบบการแสดงผลของเว็บไซต์ได้ โดย CSS จะช่วยให้เราสามารถกำหนดสี ขนาด รูปแบบตัวอักษร รวมถึงการจัดวางเนื้อหาในเว็บไซต์ได้อย่างหลากหลาย

<CodeBlock>
  {`.container {  
  display: grid;  
  grid-template-columns: repeat(3, 1fr);  
  grid-gap: 20px;  
  }
`}
</CodeBlock>
