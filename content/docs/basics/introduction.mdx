## พื้นฐาน CSS

พื้นฐานของ CSS <TextCode>color</TextCode> จะมีลักษณะการใช้งานที่ค่อนข้างง่าย โดยจะประกอบไปด้วย Selector, Property และ Value ซึ่งจะช่วยให้เราสามารถกำหนดรูปแบบการแสดงผลของเว็บไซต์ได้อย่างหลากหลาย ไม่ว่าจะเป็นการกำหนดสี ขนาด รูปแบบตัวอักษร รวมถึงการจัดวางเนื้อหาในเว็บไซต์ได้อย่างหลากหลาย

### Syntax

Syntax ของ CSS จะประกอบไปด้วย 3 ส่วนหลัก ได้แก่ Selector, Property และ Value โดยมีรูปแบบดังนี้

```css
selector {
  property: value;
}
```
