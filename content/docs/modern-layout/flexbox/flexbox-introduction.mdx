## Flexbox คืออะไร?

Flexbox เป็นหนึ่งใน layout mode ใน CSS เหมือนกับ Flow Layout หรือ Positioned Layout โดย Flexbox จะมีหน้าที่ควบคุม Element ต่างๆ ตามแกนหลัก ซึ่งแกนหลักก็สามารถเป็นได้ทั้งแนวตั้ง(column)และแนวนอน(row)

## แนวคิดพื้นฐานของ Flexbox

### One-dimensional Layout Concept

Flexbox ถูกออกแบบมาเพื่อจัดการ layout ในแนวเดียว (One-dimensional) ซึ่งหมายความว่า Flexbox จะเน้นที่การจัดเรียงองค์ประกอบในแนวแกนเดียวเท่านั้น ไม่ว่าจะเป็นแนวนอน (row) หรือแนวตั้ง (column)

เมื่อเราใช้ Flexbox เราจะกำหนด “main axis” ซึ่งเป็นแกนหลักสำหรับจัดเรียงทุก ๆ element ที่อยู่ใน container นั้น ๆ ซึ่งจะมีทิศทางเดียว ไม่ว่าจะเป็นในแนวตั้งหรือแนวนอน

### Main Axis vs Cross Axis

Flexbox ใช้แนวคิดของแกนสองแกน โดยเรียกว่า Main Axis และ Cross Axis ซึ่งช่วยให้เราเข้าใจวิธีการจัดวาง Flex Items ภายใน Container ได้ชัดเจนมากขึ้น

- **Main Axis (แกนหลัก):**
  - กำหนดโดยค่า <TextCode>flex-direction</TextCode>
  - เมื่อ <TextCode>flex-direction</TextCode> เป็น row → Main Axis คือแนวนอน (จากซ้ายไปขวา)
  - เมื่อ <TextCode>flex-direction</TextCode> เป็น column → Main Axis คือแนวตั้ง (จากบนลงล่าง)
  - ควบคุมการจัดเรียงในแกนหลักด้วยค่าเช่น justify-content
- **Cross Axis (แกนรอง):**
  - เป็นแกนที่ตั้งฉากกับ Main Axis
  - เมื่อ Main Axis เป็นแนวนอน → Cross Axis คือแนวตั้ง
  - เมื่อ Main Axis เป็นแนวตั้ง → Cross Axis คือแนวนอน
  - ควบคุมการจัดเรียงในแกนรองด้วยค่าเช่น align-items

### Flex Container และ Flex Items

Flex Container คือ element หลักหรือ Parent ที่ใช้กำหนดว่าองค์ประกอบลูกภายในนั้นจะถูกจัดเรียงโดย Flexbox

<strong>คุณสมบัติที่สำคัญของ Flex Container:</strong> ต้องมีการตั้งค่า <TextCode>
  display: flex;
</TextCode> หรือ <TextCode>display: inline-flex;</TextCode> เพื่อเปิดใช้งาน
Flexboxกำหนดทิศทางการจัดวาง (flex-direction) และการห่อหุ้ม (flex-wrap)
ขององค์ประกอบลูกได้รับผิดชอบการควบคุมตำแหน่งโดยใช้คุณสมบัติอย่าง justify-content (จัดตำแหน่งในแกนหลัก)
และ align-items (จัดตำแหน่งในแกนรอง)

<CodeBlock copiable={true} index={1}>
  {`.flex-container {  
    display: flex;  
    flex-direction: row;    /* กำหนดให้เรียงในแกนแนวนอน */  
    flex-wrap: wrap;        /* อนุญาตให้องค์ประกอบลูกห่อหุ้มบรรทัดถ้าไม่พอดีในบรรทัดเดียว */  
    justify-content: center;  
    align-items: center;  
}
`}
</CodeBlock>

Flex Container เป็นจุดเริ่มต้นที่สำคัญในการสร้าง layout ด้วย Flexbox เพราะมันจะกำหนดว่าทุกองค์ประกอบลูกจะมีการจัดเรียงและปรับขนาดแบบยืดหยุ่นได้ตามที่เราต้องการ

Flex Items คือองค์ประกอบลูกทั้งหมดภายใน Flex Container

<strong>คุณสมบัติที่สามารถปรับแต่งได้สำหรับ Flex Items:</strong>

- <TextCode>flex-grow:</TextCode> กำหนดว่า item
  นั้นจะขยายความกว้างได้มากน้อยแค่ไหนเมื่อพื้นที่ใน container มีเหลือ

- <TextCode>flex-shrink:</TextCode> กำหนดว่า item นั้นจะหดตัวลงเมื่อพื้นที่ใน
  container จำกัด

- <TextCode>flex-basis:</TextCode> กำหนดขนาดเริ่มต้นของ item
  ก่อนการขยายหรือหดตัว

- <TextCode>flex:</TextCode> เป็น shorthand ที่รวม flex-grow, flex-shrink และ
  flex-basis เข้าด้วยกัน

<CodeBlock copiable={true} index={2}>
  {`.flex-item {  
  flex-grow: 1;      /* อนุญาตให้ item ขยายตัวได้เท่า ๆ กัน */  
  flex-shrink: 1;    /* อนุญาตให้ item หดตัวลงเมื่อพื้นที่ไม่เพียงพอ */  
  flex-basis: auto;  /* ขนาดเริ่มต้นจะถูกกำหนดโดยเนื้อหาหรือค่าอื่นที่มีอยู่ */  
  /* หรือใช้ shorthand */  
  flex: 1 1 auto;    
}
`}
</CodeBlock>

การทำงานร่วมกันระหว่างคุณสมบัติของ Flex Container และ Flex Items เป็นสิ่งสำคัญที่จะทำให้ layout เป็นไปตามที่คาดหวัง

- <strong>สำหรับ Flex Container:</strong>คุณสมบัติอย่าง
  <TextCode>flex-direction</TextCode> , <TextCode>flex-wrap</TextCode>
  และ<TextCode>justify-content</TextCode>มีผลต่อการวางตำแหน่งของ Flex Items
  ในระดับ container
- <strong>สำหรับ Flex Items:</strong>
  ค่า<TextCode>flex-grow</TextCode> , <TextCode>flex-shrink</TextCode> และ
  <TextCode>flex-basis</TextCode>
  สามารถปรับแต่งให้ทุก item
  มีการปรับตัวตามขนาดที่แตกต่างกันเพื่อให้พอดีกับพื้นที่ใน containerการใช้
  shorthand flex ทำให้โค้ดดูเรียบร้อยและเข้าใจง่ายขึ้น
